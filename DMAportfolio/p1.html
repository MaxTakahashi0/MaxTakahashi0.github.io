Project 1

(Playing with Perspective/Fear and Delight) This project attempts to convey ideas of multipoint perspective and 3D shapes in a 2D picture plane. It is themed around horror manga inspired by Junji Ito, juxtaposed with bright colors.
I learned a lot about using counters and creating custom functions. My biggest challenge of making my drawing program was making sure the shapes were drawn in a way that is satisfying to the mouse movements while also retaining some unpredictability.
